Security & Machine Learning

We are based in the Department of Computing at Imperial College London, and we are interested in the cross-fertilisation of security and machine learning. In particular, we seek to address two broad research questions:

  • Can we use machine learning to address concrete security problems in novel ways?
  • Can we make machine learning system more robust and trustworthy using techniques from security and formal methods?
